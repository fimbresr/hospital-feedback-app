# Assets del Directorio Public

Este directorio contiene los assets estáticos de la aplicación.

## Archivos requeridos

Para que la aplicación funcione correctamente, necesitas agregar:

- `logo-hsda.png` - Logo del Hospital San Diego de Alcalá
- `qr-hospital.png` - Código QR de acceso a la aplicación

Estos archivos PNG no se incluyeron en el repositorio por ser binarios.
Puedes obtenerlos de la aplicación desplegada en Vercel.
